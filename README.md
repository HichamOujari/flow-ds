1- Technologie utilises:
    - Backend : 
        - mysql database
        - typeorm
        - nestjs
        - Jest

    - Frontend:
        - ReactJs
        - Chart.js

2- To run project follow those steps : 
    running backend : 
        - setup the database config in .env file
        - npm i
        - npm run import-provided-data
        - npm run start:dev

    running test :
        - npm run test

    running frontend :
        - npm i
        - npm start
